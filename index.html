01 / INGRESS

The door is outbound-only

cloudflared tunnel run personals3

Every byte enters through a Cloudflare Tunnel — a persistent connection the homelab dials out to the edge. The machine itself opens zero inbound ports: no exposed IP, no firewall holes, nothing to scan. TLS ends at the edge; what comes down the pipe is already vetted traffic.

02 / CONTROL

One Go binary decides everything

curl -H "Authorization: AWS4-HMAC-SHA256 …"

The API speaks three dialects of trust — AWS SigV4 for S3 tooling, API keys for scripts, JWT for the dashboard — and checks quota atomically before a single byte lands.

The amber glint is the rate-limit round trip to Valkey. The ice-blue pulse is metadata committing to PostgreSQL — every object is a row before it is a file.

03 / PERSISTENCE

Bytes land atomically or not at all

write → fsync → rename

Objects live content-addressed on disk. Uploads stream into a temp file, fsync, then one atomic rename — a power cut cannot leave half a file pretending to be whole. Quota is reserved in a single conditional UPDATE before the write begins.

The tank's level is this disk's real fill percentage.

04 / TRANSCODE

The furnace runs on SKIP LOCKED

SELECT … FOR UPDATE SKIP LOCKED

Upload a video and an FFmpeg worker claims the job straight out of PostgreSQL — no broker, no queue service, just a row lock. VA-API silicon burns it into an adaptive HLS ladder while progress heartbeats back every second.

The ring is live job progress. The burst is a finished encode.

05 / DELIVERY

Streams never wake the API

sendfile(2) → Cloudflare edge

Finished segments are served straight off disk by nginx — zero-copy sendfile with immutable cache headers, so the edge keeps replaying a popular video worldwide while the Go process sleeps through the whole show.

06 / INTEGRITY

The janitor trusts nothing

cleaner --interval 30s

A separate service walks an adaptive Merkle trie over the disk, comparing reality against the database — orphaned files get two strikes, then they're reaped. Verification cost grows with churn, not with how much is stored.

When the drone stops and pulls motes off the floor, that is real space being reclaimed.
